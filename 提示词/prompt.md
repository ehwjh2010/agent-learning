# 提示工程

## 公式
角色 + 任务 + 背景 + 要求 + 输出格式

### 其实这部分工作可以让 AI 给你完成，只需要给出具体的要求。
给我生成1段提示词，具体的内容如下
```
我想做让AI{...}, 帮我生成1端提示词，格式要求：角色 + 任务 + 背景 + 要求 + 输出格式
```


```
  假设你想让 AI 教你“JavaScript 函数”。

  请写一个完整提示，要求 AI：

  - 把你当成编程初学者
  - 用生活例子解释
  - 给一段简单代码
  - 逐行解释代码
  - 给 3 道练习题
  - 用表格总结重点
```
输出的格式要求必须满足：角色 + 任务 + 背景 + 要求 + 输出格式

## 示例
比如你想让 AI 帮你学英语，不要只写：</br>

教我英语

可以写成：

你是一位耐心的英语老师。
我是英语初学者，词汇量大约 1000。
请教我今天的 5 个常用动词。
每个词请包含：中文意思、英文例句、中文翻译、一个记忆方法。
请用表格输出。
